## Troubleshooting

### iOS: Blank Screen
```bash
# Clean build
# In Xcode: Product â†’ Clean Build Folder (Cmd+Shift+K)

# Delete derived data
rm -rf ~/Library/Developer/Xcode/DerivedData/iosApp-*

# Rebuild
cd iosApp
xcodebuild -project iosApp.xcodeproj \
  -scheme iosApp \
  -configuration Debug \
  -sdk iphonesimulator \
  -arch arm64 \
  build
```

### Android: Blank Screen
```bash
# Clean build
./gradlew clean

# Rebuild
./gradlew :androidApp:installDebug

# Clear app data
adb shell pm clear com.eunio.healthapp.android
```
